# 课程总结

## nodejs
* 实现用于后端开发，选用JS语言
* 掌握语言和实际开发有很大差异
* 后端开发和前端开发有着不同的设计和思路
* nodejs中的stream、fs等均是server端的概念
* nodejs并不像原型异步等，隶属于JS的模块
* nodejs是一个使用了JS语法的独立的技术栈
